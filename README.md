**Weather App**
A modern and user-friendly web app that shows the current weather for any city using the OpenWeatherMap API.
Built with HTML, CSS, and JavaScript, it’s fully interactive and beginner-friendly.

Features:
*Search for any city to get real-time weather information
*Displays temperature, weather condition, humidity
*Dynamic weather icons that blend with the background
*Glassy, aesthetic UI with glassmorphism effects
*Easy to customize and expand

How to Run:
1.Clone the repository->
    git clone https://github.com/arurihelona/Weather-App.git
2.Open the folder in VS Code (or any editor).
3.Open index.html in a browser (or use the Live Server extension).
4.Enter a city name → click Get Weather.
  Make sure to replace the placeholder API key in script.js with your own OpenWeatherMap API key.

API Used:
*OpenWeatherMap API(free)
